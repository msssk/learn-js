Web browsers can load JavaScript code from a file and run it. They can also run code live as you type it.

Press F12 to open the developer tools, then click on the Console tab.

You can type code directly, hit Enter, and it will be executed. Try some simple expressions:

  • 1
  • 1 + 1
  • 'green'
  • green

You should have gotten an error after that last one. This is because JavaScript doesn't know what green is. But you can tell JavaScript what green is! Try the following:

  • green = 5
  • green

You've just defined the variable green. You can now use it:

  • green + 1
JavaScript has a set of defined words that can be used in your code, but can't be used as variable names. JavaScript keywords

So far you've done very simple things. Information and values are collectively referred to as "data" in programming. The kind of data you are storing is referred to as a "data type". In JavaScript there are a few simple data types known as "primitive data types:"

  1. string: Examples: "green", "Name", "A full sentence can be a string"
  2. number: Examples: 1, 5, -5000, 1e5
  3. boolean: Examples: true, false
This is good to get you started, but there's a lot more to learn about JavaScript data types.
Reviewing what you've done so far, you should notice a few things:
  1. JavaScript has operators, you've already used + and =
  2. When you want to create some data you can store it in a "variable"
  3. Variables are assigned a value with the = operator
